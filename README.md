# Chat4Us-Setup

![Java](https://img.shields.io/badge/Java-21%2B-orange?style=for-the-badge&logo=openjdk)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-blue?style=for-the-badge)

Chat4Us-Setup is a lightweight Java-based installer application designed to manage the complete lifecycle of Chat4Us projects. This setup utility provides an automated solution for installing, updating, recovering and uninstalling Chat4Us projects binaries across all major platforms.

Project Homepage: [chat4usai.com](https://chat4usai.com)

Ensure you have Java 21 or later installed on your system.
